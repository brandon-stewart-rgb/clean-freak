## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/tyler1836/workout-tracker
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
